Sandpile Simulation

Simulation of the so called Bak-Tang-Wiesenfeld model (or Abelian Sandpile Model). Each square can hold up to 4 grains of sand, if it gets to 4, it spills them on the surroinding neighbours. The term self-organized criticality was introduced to describe this model. The lattice is initialized randomly with some sand, then each frame a new grain is dropped at random on one site. Clicking with the mouse will drop 1000 grains on that site.
